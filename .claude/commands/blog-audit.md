# /blog-audit — 블로그 글 품질 감사

description: 기존 블로그 글을 브랜드 프로필 기준으로 품질 감사

## 사용법
```
/blog-audit <brand-slug> | <파일경로>
```
예시: `/blog-audit sos-coffee-brunch | output/sos-coffee-brunch/blog/20260306_석촌역맛집.md`

---

## 실행 절차

### 0. ARGUMENTS 파싱
`$ARGUMENTS`를 파이프(`|`) 기준으로 분리한다.
- 파이프 앞: `BRAND_SLUG`
- 파이프 뒤: `FILE_PATH` (감사 대상 파일 경로)

두 값 중 하나라도 비어 있으면 아래 안내 후 중단:
```
사용법: /blog-audit <brand-slug> | <파일경로>
예시:   /blog-audit sos-coffee-brunch | output/sos-coffee-brunch/blog/20260306_석촌역맛집.md
```

---

### 1. 브랜드 프로필 로딩
다음 경로에서 브랜드 파일을 탐색한다 (순서대로):
1. `brands/BRAND_SLUG.md`
2. `brands/BRAND_SLUG/profile.md`
3. `brands/` 하위에서 파일명에 `BRAND_SLUG`가 포함된 `.md` 파일

**매칭 실패 시** 아래 메시지를 출력하고 중단:
```
브랜드 파일을 찾을 수 없습니다: BRAND_SLUG
brands/_template.md 를 복사해 브랜드 프로필을 먼저 작성해 주세요.
```

---

### 2. 대상 파일 읽기
`FILE_PATH`에서 파일을 읽는다.

파일이 없을 경우:
```
파일을 찾을 수 없습니다: FILE_PATH
경로를 확인해 주세요.
```

---

### 3. 100점 체크리스트 채점

아래 항목별로 점수를 매기고 감점 사유를 기록한다.

#### A. SEO 최적화 (30점)
| 항목 | 배점 | 득점 | 감점 사유 |
|------|------|------|-----------|
| 제목에 핵심 키워드 포함 | 5 | | |
| 핵심 키워드 5회 이상 등장 | 5 | | |
| 관련 키워드 자연 분산 배치 | 5 | | |
| 메타 설명 / 도입부 키워드 포함 | 5 | | |
| 해시태그 10개 이상 | 5 | | |
| 소제목에 키워드 반영 | 5 | | |

#### B. 글 구조 & 가독성 (20점)
| 항목 | 배점 | 득점 | 감점 사유 |
|------|------|------|-----------|
| 전체 글자수 1,500자 이상 | 5 | | |
| 소제목 4개 이상 | 5 | | |
| 도입부·본문·마무리 구조 완성 | 5 | | |
| 문단 길이 적절 (5줄 이내 권장) | 5 | | |

#### C. 브랜드 정합성 (25점)
| 항목 | 배점 | 득점 | 감점 사유 |
|------|------|------|-----------|
| 브랜드 톤앤매너 일치 | 10 | | |
| 금지 표현 0건 | 10 | | |
| 필수 포함 정보 누락 없음 | 5 | | |

#### D. 정보 정확성 (15점)
| 항목 | 배점 | 득점 | 감점 사유 |
|------|------|------|-----------|
| 브랜드 프로필에 없는 정보 지어냄 | -15 (위반시) | | |
| 영업시간·가격·메뉴 등 사실 정보 정확 | 10 | | |
| 위치·교통 정보 정확 | 5 | | |

#### E. CTA & 전환 (10점)
| 항목 | 배점 | 득점 | 감점 사유 |
|------|------|------|-----------|
| CTA 1개 이상 포함 | 5 | | |
| CTA 위치·문구 적절성 | 5 | | |

### 4. 감사 결과 출력

```
=== 블로그 글 품질 감사 결과 ===
대상 파일: FILE_PATH
브랜드: BRAND_SLUG
감사 일시: YYYY-MM-DD

총점: N / 100점

[등급]
90~100: S (즉시 발행 가능)
80~89:  A (경미한 수정 후 발행)
70~79:  B (보통 수정 필요)
60~69:  C (대폭 수정 필요)
~59:    D (재작성 권장)

[항목별 점수]
A. SEO 최적화:      N/30
B. 글 구조 & 가독성: N/20
C. 브랜드 정합성:   N/25
D. 정보 정확성:     N/15
E. CTA & 전환:      N/10
```

항목별 감점 사유와 구체적인 수정 제안을 함께 출력한다.

**브랜드 프로필에 없는 정보 포함 여부**를 별도 섹션으로 강조 표시:
```
[⚠️ 확인되지 않은 정보]
- (해당 없으면 "없음"으로 표시)
- 발견 시: 해당 문장 인용 + 수정 방향 제안
```

---

### 5. 파일 저장
경로: `./output/BRAND_SLUG/audits/YYYYMMDD_키워드슬러그_audit.md`

- `YYYYMMDD`: 오늘 날짜
- `키워드슬러그`: 대상 파일명에서 날짜 부분을 제외한 슬러그
- 저장 완료 후 경로 안내 및 점수에 따른 다음 행동 제안:
  - 80점 미만: `/blog BRAND_SLUG | KEYWORD` 로 재작성 권장
  - 80점 이상: 수정 포인트만 적용 후 발행 권장
