# /blog-plan — 블로그 콘텐츠 기획안 생성

description: 브랜드 프로필 기반 블로그 콘텐츠 기획안 생성

## 사용법
```
/blog-plan <brand-slug> | <키워드>
```
예시: `/blog-plan sos-coffee-brunch | 잠실브런치카페`

---

## 실행 절차

### 0. ARGUMENTS 파싱
`$ARGUMENTS`를 파이프(`|`) 기준으로 분리한다.
- 파이프 앞: `BRAND_SLUG`
- 파이프 뒤: `KEYWORD`

두 값 중 하나라도 비어 있으면 아래 안내 후 중단:
```
사용법: /blog-plan <brand-slug> | <키워드>
예시:   /blog-plan sos-coffee-brunch | 잠실브런치카페
```

---

### 1. 브랜드 프로필 로딩
다음 경로에서 브랜드 파일을 탐색한다 (순서대로):
1. `input/brands/BRAND_SLUG.md`
2. `input/brands/BRAND_SLUG/profile.md`
3. `input/brands/` 하위에서 파일명에 `BRAND_SLUG`가 포함된 `.md` 파일

**매칭 실패 시** 아래 메시지를 출력하고 중단:
```
브랜드 파일을 찾을 수 없습니다: BRAND_SLUG
input/brands/_template.md 를 복사해 브랜드 프로필을 먼저 작성해 주세요.
```

---

### 2. 글 유형 판별
브랜드 프로필의 **유형별 키워드 목록**과 `KEYWORD`를 대조해 글 유형을 결정한다.

- 매칭 불가 시 사용자에게 확인:
  ```
  키워드 "KEYWORD"에 해당하는 글 유형을 판별하지 못했습니다.
  브랜드 프로필에 등록된 유형: [유형 목록]
  어떤 유형으로 기획할까요?
  ```

---

### 3. 키워드 분석
- 타깃 키워드의 검색 의도 분류 (정보형 / 탐색형 / 거래형)
- 월간 검색량 추정 레벨 (높음 / 중간 / 낮음)
- 연관 롱테일 키워드 10개 도출 및 우선순위 정렬
- 시즌성·트렌드 요소 파악

---

### 4. 경쟁 분석
- 해당 키워드로 상위 노출될 가능성이 높은 콘텐츠 유형 분석
- 경쟁 콘텐츠와 차별화할 수 있는 앵글 3가지 제안
- 브랜드 강점을 부각할 수 있는 포인트

---

### 5. 기획안 작성

#### 제목 3안
각 제목은:
- 핵심 키워드 포함
- 35자 이내
- 클릭 유도 요소 포함 (숫자, 질문형, 혜택 강조 등)
- 서로 다른 앵글로 작성

#### 도입부 방향
- 독자의 검색 의도를 정확히 반영한 첫 문장 전략
- 2~3문장 분량의 도입부 샘플

#### 소제목 구성안
- 소제목 4~6개 및 각 섹션에서 다룰 핵심 포인트
- 키워드 자연 삽입 위치 표시

#### CTA 방향
- 브랜드 프로필의 CTA 지침 기반
- 본문 중간 CTA + 마무리 CTA 위치 및 문구 제안

---

### 6. 파일 저장
경로: `./output/BRAND_SLUG/plans/YYYYMMDD_키워드슬러그_plan.md`

- `YYYYMMDD`: 오늘 날짜
- `키워드슬러그`: KEYWORD를 소문자·하이픈 형태로 변환 (한글 그대로 사용 가능)
- 저장 완료 후 경로와 함께 다음 단계 안내:
  ```
  기획안이 저장되었습니다: [경로]
  이 기획안으로 바로 글을 작성하려면:
  /blog BRAND_SLUG | KEYWORD
  ```
