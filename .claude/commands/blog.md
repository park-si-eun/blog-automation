# /blog — 네이버 블로그 SEO 최적화 글 생성

description: 브랜드 프로필 기반 네이버 블로그 SEO 최적화 글 생성

## 사용법
```
/blog <brand-slug> | <키워드>
```
예시: `/blog sos-coffee-brunch | 석촌역맛집`

---

## 실행 절차

### 0. ARGUMENTS 파싱
`$ARGUMENTS`를 파이프(`|`) 기준으로 분리한다.
- 파이프 앞: `BRAND_SLUG` (브랜드 파일명 또는 브랜드명)
- 파이프 뒤: `KEYWORD` (타깃 키워드)

두 값 중 하나라도 비어 있으면 아래 안내 후 중단:
```
사용법: /blog <brand-slug> | <키워드>
예시:   /blog sos-coffee-brunch | 석촌역맛집
```

---

### 1. 브랜드 프로필 로딩
다음 경로에서 브랜드 파일을 탐색한다 (순서대로):
1. `input/brands/BRAND_SLUG.md`
2. `input/brands/BRAND_SLUG/profile.md`
3. `input/brands/` 하위에서 파일명에 `BRAND_SLUG`가 포함된 `.md` 파일

**매칭 실패 시** 아래 메시지를 출력하고 중단:
```
브랜드 파일을 찾을 수 없습니다: BRAND_SLUG
input/brands/_template.md 를 복사해 브랜드 프로필을 먼저 작성해 주세요.
```

브랜드 파일을 찾으면 전체 내용을 읽어 이후 단계에서 참조한다.

---

### 2. 글 유형 판별
브랜드 프로필의 **유형별 키워드 목록**과 `KEYWORD`를 대조해 글 유형을 결정한다.

- 매칭되는 유형이 1개면 해당 유형으로 확정
- 매칭 불가 시 사용자에게 확인:
  ```
  키워드 "KEYWORD"에 해당하는 글 유형을 판별하지 못했습니다.
  브랜드 프로필에 등록된 유형: [유형 목록]
  어떤 유형으로 작성할까요?
  ```

---

### 3. STEP 0 — 사전 조사
- 타깃 키워드의 검색 의도 파악 (정보형 / 탐색형 / 거래형)
- 관련 롱테일 키워드 5~10개 도출
- 브랜드 프로필에서 이 글에 활용 가능한 정보 목록화

### 4. STEP 1 — 제목 설계
- 검색 의도에 맞는 제목 3안 작성
- 각 제목: 핵심 키워드 포함, 35자 이내, 클릭 유도 요소 포함
- 최종 제목 1안 확정 (또는 사용자 선택 요청)

### 5. STEP 2 — 구조 설계
- 도입부 / 본문 소제목 4~6개 / 마무리 구성
- 각 소제목 아래 다룰 핵심 내용 한 줄 메모
- 키워드 배치 계획 (제목·도입부·소제목·본문·마무리)

### 6. STEP 3 — 초안 작성
브랜드 프로필의 톤앤매너·금지 표현·필수 포함 정보를 준수하며 초안을 작성한다.
- 전체 1,500자 이상 (권장 2,000~3,000자)
- 핵심 키워드: 제목 포함 5회 이상
- 관련 키워드: 자연스럽게 분산 배치
- 브랜드 프로필에 없는 정보는 절대 지어내지 않는다

### 7. STEP 4 — SEO 최적화
- 메타 설명 작성 (160자 이내)
- 이미지 ALT 텍스트 제안 3~5개
- 내부 링크 삽입 위치 제안
- 해시태그 10~15개 생성

### 8. STEP 5 — 톤 & 브랜드 정합성 검토
- 브랜드 프로필의 금지 표현 제거
- 브랜드 보이스에 맞게 문체 다듬기

### 9. STEP 6 — CTA 삽입
- 브랜드 프로필의 CTA 지침에 따라 적절한 위치에 삽입
- CTA 유형: 방문 유도 / 예약 유도 / 공유 유도 등

### 10. STEP 7 — 최종 완성
완성 글 전문 출력

---

### 11. 자가 체크리스트
글 생성 후 아래 항목을 자동 점검하고 결과를 표로 출력한다:

| 항목 | 기준 | 결과 |
|------|------|------|
| 글자수 | 1,500자 이상 | ✅/❌ (실제 N자) |
| 핵심 키워드 횟수 | 5회 이상 | ✅/❌ (실제 N회) |
| CTA 포함 | 1개 이상 | ✅/❌ |
| 금지 표현 | 0건 | ✅/❌ (위반 N건) |
| 없는 정보 지어냄 | 없어야 함 | ✅/❌ |

점검 항목 중 ❌가 있으면 즉시 수정 후 재출력한다.

---

### 12. 파일 저장
경로: `./output/BRAND_SLUG/blog/YYYYMMDD_키워드슬러그.md`

- `YYYYMMDD`: 오늘 날짜
- `키워드슬러그`: KEYWORD를 소문자·하이픈 형태로 변환 (한글 그대로 사용 가능)
- 파일 저장 후 경로를 사용자에게 안내한다
